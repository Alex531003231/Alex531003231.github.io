— 凡是涉及到into查询语句赋值或者 := 赋值操作，都必须使用out来修饰

存储过程和存储函数的区别*

  1. 语法区别
    • 关键字不一样
    • 存储函数比存储过程多了两个return
  2. 本质区别
    • 存储函数有返回值，而存储过程没有返回值
    • 如果存储过程想实现有返回值的业务，我们就必须使用out类型的参数
    • 即使存储过程使用了out类型的参数，其本质也不是真的有了返回值
    • 而是在存储过程内部给了out类型参数赋值，在执行完毕后，我们直接拿到了输出类型参数的值
  3. 我们可以使用存储函数有返回值的特性，来自定义函数
  4. 存储过程不能用自定义函数

触发器介绍：

– 触发器就是制定一个规则，在我们做增删改操作的时候

—-只要满足该规则，自动触发，无需调用

—-语句级触发器:不包含for each row 的触发器

—-行级触发器：包含for each row 的就是行级触发器
